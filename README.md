## Services

### Pre-audit code review — Available now · $4K–$8K · 5–7 days 

A focused security review by a senior engineer who has shipped production smart contracts handling real money. Designed to be done **before** you pay for a full audit, so you don't waste $20K+ on a remediation round catching obvious issues.

**Scope:** up to 1500 nSLOC · Solidity / EVM only

**Deliverable:** PDF report with findings classified by severity (Critical / High / Medium / Low / Informational + Gas optimizations), reproduction notes, and suggested fixes. Includes one 30-minute call to walk through the report.

**What I look for:** input validation, access control, oracle and price-feed assumptions, rounding and precision in liquidity math, reentrancy and external-call patterns, accounting consistency in dynamic LP pools, role and permission design, common patterns that have led to recent exploits.

**What's not included:** formal verification, economic / game-theoretic modeling, off-chain or infrastructure security, re-review after fixes (available separately for $1K).

> A pre-audit review is not a substitute for a full security audit by a specialized firm. The protocol team retains full responsibility for the security of deployed contracts.
